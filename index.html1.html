Questionário

a. Na tag < input > com atributo type="text", para que serve o atributo "placeholder"? O atributo "placeholder" em um elemento < input > com o atributo "type" definido como "text" é usado para fornecer um texto de exemplo ou uma dica dentro do campo de entrada. Ele serve para indicar aos usuários qual tipo de informação ou formato é esperado no campo antes de eles inserirem seus próprios dados. O texto do atributo "placeholder" desaparece assim que o usuário começa a digitar no campo. Isso ajuda a melhorar a usabilidade de formulários em sites e aplicativos, fornecendo orientações visuais aos usuários. Por exemplo: <input type="text" placeholder="Digite seu nome">
b. Na tag < input >> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"? O atributo "checked" em elementos < input >> com o atributo "type" definido como "radio" ou "checkbox" serve para indicar se um botão de opção (radio button) ou uma caixa de seleção (checkbox) está marcado (selecionado) por padrão quando a página é carregada. Se o atributo "checked" estiver presente e definido como "checked", o botão de opção ou a caixa de seleção será exibido como selecionado quando a página for carregada. Se estiver ausente ou definido como vazio, o elemento será exibido como desmarcado por padrão. Isso permite que você controle o estado inicial desses elementos de entrada em um formulário HTML.
c. Na tag tag < input > para que serve o atributo "value"? O atributo "value" em uma tag < input > em HTML é usado para especificar o valor inicial ou o valor padrão que deve aparecer na caixa de entrada (input) quando a página é carregada. Este atributo permite que você defina um valor predefinido para o campo de entrada, que pode ser útil em formulários ou sempre que você desejar fornecer um valor inicial para os usuários
d. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez? Para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez, você deve usar o atributo "name" com o mesmo valor para todos os botões de rádio que deseja agrupar. O atributo "name" é usado para identificar um grupo de botões de rádio e garantir que apenas um deles possa ser selecionado ao mesmo tempo.
e. Para que serve a tag < label >? Para que serve o atributo "for" nessa tag? A tag < label > em HTML é utilizada para associar um rótulo descritivo a um elemento de formulário,como <input>. Isso melhora a usabilidade do formulário,permitindo que os usuários saibam qual informação deve ser inserida em um campo específico
f. Na tag < option > para que serve o atributo "selected"? O elemento HTML select (<select>) representa um controle que apresenta um menu de opções. As opções dentro do menu são representadas pelo elemento <option> , que podem ser agrupados por elementos <optgroup> . As opções podem ser pré-selecionadas para o usuário.

Quando um campo é marcado como "required", o navegador do usuário verificará automaticamente se ele foi preenchido antes de permitir o envio do formulário.
h. Para que serve a tag < textrea >? A tag <textarea> em HTML é usada para criar uma área de texto expansível em um formulário ou página da web. Ela permite que os usuários insiram texto em várias linhas e é útil quando você deseja coletar informações mais extensas, como comentários, mensagens longas ou outros tipos de entrada de texto que podem ser mais do que apenas uma linha única. Aqui está um exemplo de como você pode usar a tag <textarea> em HTML: <textarea rows="4" cols="50"> Aqui você pode digitar várias linhas de texto. </textarea>
i. Na tag < form > para que serve o atributo "action"? O atributo "action" na tag < form > em HTML especifica o URL para o qual os dados do formulário serão enviados quando o formulário for submetido. O navegador da web enviará os dados do formulário para o URL especificado no atributo "action" quando o usuário pressionar o botão de envio (geralmente o botão "Submit").
j. Na tag < form > para que serve o atributo "method"? O atributo "method" em HTML é utilizado na tag < form > para especificar o método HTTP que será utilizado quando o formulário for submetido para o servidor web. O atributo "method" possui dois valores principais
k. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form >? Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, o atributo "method" na tag < form > não é o fator determinante. O atributo "method" em HTML especifica como os dados do formulário serão enviados para o servidor, e as opções comuns são "GET" e "POST". No entanto, isso não afeta diretamente a visibilidade da senha na barra de endereços. A visibilidade da senha na barra de endereços do navegador pode ocorrer principalmente devido à forma como os dados são enviados (por exemplo, como parte da URL) ou devido a scripts JavaScript maliciosos ou vulnerabilidades de segurança no site.
l. Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos? No atributo “method” na tag < form > em um formulário usado para fazer upload de arquivos,o valor que dever ser informado é “POST”
m. O que as tags < fieldset > e < legend > produzem? As tags HTML <fieldset> e > legend > são usadas em conjunto para criar um grupo de elementos de formulário e fornecer uma legenda ou título para esse grupo.
